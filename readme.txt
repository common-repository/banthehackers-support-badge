== Description ==

Show your support for BanTheHackers with a drop down badge. The badge uses minimal real estate showing only a small red bookmark icon on the top right of your website. When a user mouses over this the badge drops down with the "We Support BanTheHackers" message. The message if clicked links your visitors through to [BanTheHackers](http://BanTheHackers.tk/)


== Installation ==

1. Upload `BanTheHackers.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is BanTheHackers? =

BanTheHackers.tk is a website aimed at exposing hackers in MapleStory, a game published by Nexon.

= Do you hate hackers? =

Yes, yes we do.
